## ğŸ§  Puzzle Generation (Intelligent Contract Integration)

Puzzle content is generated by an **intelligent contract** that:
- Uses AI to create mythology puzzles
- Selects unique gods from approved pantheons
- Produces structured puzzle data:
  - Correct answer
  - Pantheon
  - What the god is known for
  - Three progressive hints

The frontend:
- Requests a new puzzle set at the start of a game
- Reads the generated puzzle data
- Handles all gameplay logic locally (hint reveals, guesses, scoring)

The blockchain layer is **intentionally invisible** to players.
